# 债券明星经理专题

本专题用于沉淀债券/固收+基金经理对比研究，优先记录可直接用于配置决策的结论，包括：

- 当前在管基金清单（同基金不同份额去重）
- 近5年收益或成立以来收益
- 经理风格与代表产品
- 调仓参考与后续跟踪事项

## 目录

- `2026-04-26.md`：张芊、过钧、刘太阳、张清华四位债券明星经理专题对比
- `TODO.md`：后续复核与调仓提醒事项

## 研究口径

- 默认只看经理**当前在管**基金
- 同一基金的 A/C/B/D/E 等份额默认保留 **A 类/主份额**
- 满 5 年的基金按**近5年收益**比较
- 成立不足 5 年的基金单独标注为**成立以来**
- 数据来源以天天基金公开页面与 `pingzhongdata/<基金代码>.js` 累计净值序列为主

## 当前结论摘要

- 过钧在当前可比的满 5 年样本中，整体弹性与收益明显领先。
- 张芊与张清华更偏稳健、均衡型固收+配置思路。
- 刘太阳当前可比样本过少，适合作为观察补充，不适合直接与前三位并列定性。
- 若用于调仓，需把经理风格、底层资产暴露、组合底仓和个人波动承受能力一起考虑。
