# 治理评价说明文档

## 1. 专题目标

本目录用于记录 A 股上市公司管理层与治理质量的阶段性评估结果，形成可追踪、可复盘的治理评价档案。

## 2. 当前纳入范围

- 市场范围：A 股上市公司
- 评价对象：公司核心管理层与治理体系
- 初始输出：A 股优秀管理层 Top50 筛选版

## 3. 评价逻辑

采用交叉验证方式，而非单一榜单判断。重点参考以下五类信号：

1. **战略与执行**
   - 是否长期跑赢同业
   - 是否具备产业升级、国际化、组织迭代能力
2. **治理与信披**
   - 公司治理结构是否规范
   - 信息披露、投资者关系、董事会运作质量是否稳定
3. **资本配置**
   - 是否具备分红纪律、稳健融资策略、理性扩张能力
4. **组织能力**
   - 高管稳定性、梯队建设、授权协同能力
5. **风险与合规**
   - 是否存在重大处罚、财务舞弊、审计非标、治理争议

## 4. 主要参考口径

- 优秀 CEO / 企业家公开榜单
- 公司治理、信披、投资者关系相关公开评价
- 长期经营质量与资本市场口碑
- 管理层稳定性与负面事件筛查

## 5. 文件规范

- 每次评估使用一个日期文件，命名格式：`YYYY-MM-DD.md`
- 文件内容建议包括：
  - 评估范围
  - 核心结论
  - Top 名单
  - 方法说明
  - 风险提示
  - 下次复盘建议
