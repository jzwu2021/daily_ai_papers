# AI Papers Digest — 2026-05-13

Selected recent arXiv papers.

## Overview

This paper set is strongly centered on agent systems, memory, evaluation, grounding, and deployability. Several papers push agents closer to real runtime conditions, while others tackle the bottlenecks that appear when models need to remember, act safely, or run efficiently on constrained hardware. Robotics and multimodal work also continue to converge around adaptation and reliable world interaction.

## Papers

### 1. Shepherd: A Runtime Substrate Empowering Meta-Agents with a Formalized Execution Trace
Date: 2026-05-11
URL: https://arxiv.org/abs/2605.10913
Summary: Shepherd formalizes meta-agent operations as functions and records agent-environment interaction as a typed, Git-like execution trace that can be replayed and forked. The paper matters because it treats intervention, reproducibility, and prompt-cache reuse as first-class runtime primitives for real agent systems rather than afterthought tooling.

### 2. Remember the Decision, Not the Description: A Rate-Distortion Framework for Agent Memory
Date: 2026-05-11
URL: https://arxiv.org/abs/2605.10870
Summary: This work argues that agent memory should be judged by how well it preserves decision quality under a fixed budget, not by how descriptively it summarizes the past. That reframing is important because long-horizon agents need compression schemes aligned with action quality, not just relevance or salience heuristics.

### 3. WildClawBench: A Benchmark for Real-World, Long-Horizon Agent Evaluation
Date: 2026-05-11
URL: https://arxiv.org/abs/2605.10912
Summary: WildClawBench introduces a native-runtime benchmark of 60 human-authored, bilingual, multimodal tasks that are longer and more realistic than common sandbox evaluations. Its importance is that it measures whether agents can sustain real work over many steps and tool interactions in the environments where they are actually deployed.

### 4. Grounded or Guessing? LVLM Confidence Estimation via Blind-Image Contrastive Ranking
Date: 2026-05-11
URL: https://arxiv.org/abs/2605.10893
Summary: The paper targets a key multimodal failure mode: vision-language models can answer fluently from language priors even when the image contributes little or nothing. By estimating confidence through contrast between normal and blind-image behavior, it offers a practical way to detect when a model is sounding grounded without actually being grounded.

### 5. DECO: Sparse Mixture-of-Experts with Dense-Comparable Performance on End-Side Devices
Date: 2026-05-11
URL: https://arxiv.org/abs/2605.10933
Summary: DECO is a sparse MoE architecture designed to preserve strong performance while reducing the storage and memory-access bottlenecks that make end-side deployment difficult. That makes it notable as a deployment-oriented contribution, especially for settings where model quality must coexist with strict device constraints.

### 6. Beyond Red-Teaming: Formal Guarantees of LLM Guardrail Classifiers
Date: 2026-05-11
URL: https://arxiv.org/abs/2605.10901
Summary: This paper argues that empirical red-teaming is not enough for production guardrail classifiers and instead proposes formal guarantees in the model’s pre-activation space. The contribution is significant because it pushes safety tooling toward certifiable behavior instead of relying only on best-effort testing coverage.

### 7. HarmoWAM: Harmonizing Generalizable and Precise Manipulation via Adaptive World Action Models
Date: 2026-05-11
URL: https://arxiv.org/abs/2605.10942
Summary: HarmoWAM studies a core tradeoff in robot world action models between broad generalization and precise interaction control, then proposes an adaptive approach to combine both strengths. This is important because robot learning systems often fail when forced to choose between transferable world understanding and fine-grained manipulation quality.

### 8. PriorVLA: Prior-Preserving Adaptation for Vision-Language-Action Models
Date: 2026-05-11
URL: https://arxiv.org/abs/2605.10925
Summary: PriorVLA keeps a frozen prior expert and trains a separate adaptation expert so downstream robot tuning does not overwrite useful pretrained knowledge. The idea matters because foundation-style robot models will only scale if they can specialize without collapsing the broad priors gained from large-scale pretraining.

## Themes

- Agent research is moving toward native-runtime evaluation, formal traces, and decision-aware memory.
- Multimodal reliability remains a central concern, especially around visual grounding and confidence.
- Practical deployment pressure is shaping model architecture work, including efficient MoE design.
- Safety research is shifting from red-team-only evaluation toward stronger formal guarantees.
- Robotics papers continue to focus on adapting foundation models without losing generalization.
