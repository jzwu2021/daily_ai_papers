# AI Papers Digest — 2026-05-12
Selected recent arXiv papers.

## Overview
Today’s paper set highlights how quickly the field is moving from larger models toward better runtime behavior, better structure, and better systems integration. Several papers focus on reasoning-time optimization, tool use, safety, and efficient generation, while others push multimodal generation, robotics, autonomous driving data infrastructure, and non-invasive brain-signal decoding. Overall, the strongest pattern is a shift from raw capability gains to controllability, efficiency, and deployment-ready abstractions.

## Papers
### 1. LLMs Improving LLMs: Agentic Discovery for Test-Time Scaling
Date: 2026-05-11
URL: https://arxiv.org/abs/2605.08083
Summary: This paper proposes AutoTTS, an environment-driven framework that searches for test-time scaling strategies instead of hand-crafting them. By casting width-depth scaling as controller synthesis over reasoning traces and probe signals, it improves the accuracy-cost tradeoff on math reasoning benchmarks and generalizes across held-out benchmarks and model scales.

### 2. Rubric-Grounded RL: Structured Judge Rewards for Generalizable Reasoning
Date: 2026-05-11
URL: https://arxiv.org/abs/2605.08061
Summary: The paper replaces binary or holistic rewards with weighted, verifiable rubric criteria scored by a frozen LLM judge using auxiliary grounding the policy never sees. Training with these structured rewards improves held-out rubric performance and transfers to GSM8K, MATH, and GPQA, making it a notable result for robust reasoning RL.

### 3. Fast Byte Latent Transformer
Date: 2026-05-11
URL: https://arxiv.org/abs/2605.08044
Summary: This work addresses the slow generation speed of byte-level language models by adding a block-wise diffusion objective that enables multiple bytes to be generated in parallel. The resulting methods cut estimated memory-bandwidth cost substantially, making byte-level modeling more plausible for efficient inference.

### 4. Tool Calling is Linearly Readable and Steerable in Language Models
Date: 2026-05-11
URL: https://arxiv.org/abs/2605.07990
Summary: Across 12 instruction-tuned models, the authors find that tool choice is linearly readable and causally steerable from internal activations. The paper matters because it suggests practical interpretability and intervention handles for improving agent tool routing instead of treating tool use as a black-box behavior.

### 5. GLiGuard: Schema-Conditioned Classification for LLM Safeguard
Date: 2026-05-11
URL: https://arxiv.org/abs/2605.07982
Summary: GLiGuard reframes safeguarding as schema-conditioned classification with a 0.3B bidirectional encoder rather than autoregressive decoding. In one non-autoregressive pass it can jointly score prompt safety, response safety, refusal, harm categories, and jailbreak strategies, showing a promising speed-quality tradeoff for production guardrails.

### 6. Object Hallucination-Free Reinforcement Unlearning for Vision-Language Models
Date: 2026-05-11
URL: https://arxiv.org/abs/2605.08031
Summary: The paper argues that unlearning only the language decoder leaves visual knowledge largely intact and can create hallucinated objects. Its HFRU framework instead targets the vision encoder and combines alignment disruption with GRPO-based optimization, achieving strong forgetting and retention with negligible object hallucination.

### 7. STARFlow2: Bridging Language Models and Normalizing Flows for Unified Multimodal Generation
Date: 2026-05-11
URL: https://arxiv.org/abs/2605.08029
Summary: The authors propose autoregressive normalizing flows as a better structural match to language models than diffusion for unified text-image generation. By interleaving a pretrained VLM stream with a TarFlow stream under the same causal mask and KV-cache, the system supports cache-friendly multimodal generation while staying strong on both image generation and understanding.

### 8. 123D: Unifying Multi-Modal Autonomous Driving Data at Scale
Date: 2026-05-11
URL: https://arxiv.org/abs/2605.08084
Summary: 123D introduces a single API for fragmented autonomous-driving datasets by representing each modality as an independent timestamped event stream. The framework unifies eight real-world datasets plus a synthetic one, which matters because shared data abstractions can lower friction for transfer learning, planning, and multimodal evaluation.

### 9. BrickCraft: Visuomotor Skill Composition with Situated Manual Guidance for Long-Horizon Interlocking Brick Assembly
Date: 2026-05-11
URL: https://arxiv.org/abs/2605.07605
Summary: BrickCraft tackles long-horizon robotic assembly by decomposing tasks into reusable primitive skills anchored to reference bricks in a partial structure. Its situated manuals connect assembly intent to real robot observations, showing a practical path toward compositional robot skill reuse with limited demonstrations.

### 10. Zero-Shot Imagined Speech Decoding via Imagined-to-Listened MEG Mapping
Date: 2026-05-11
URL: https://arxiv.org/abs/2605.08075
Summary: This work maps imagined MEG responses into a listened-speech space and then decodes them with a contrastive decoder trained only on listened data. The result is a zero-shot imagined-speech pipeline that works above chance on held-out subjects, pointing toward more scalable non-invasive brain-computer interfaces.

## Themes
- Test-time reasoning and structured RL are becoming key levers for improving performance without simply scaling model size.
- Tool use, guardrails, and unlearning are increasingly treated as interpretable and engineerable system behaviors.
- Efficient generation methods are expanding beyond standard token-level autoregression into byte-level and multimodal alternatives.
- Robotics, autonomous driving, and brain-computer interfaces continue to benefit from better abstractions that connect learning systems to real-world data and action.
