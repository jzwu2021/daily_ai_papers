# AI Papers Digest — 2026-05-05
Selected recent arXiv papers.

## Overview
The strongest recent arXiv papers focus on the operating discipline around advanced AI systems: whether models follow procedures faithfully, decide when to use tools, execute structured plans, and stay efficient under multimodal inference loads. At the same time, embodied and agent research is pushing toward longer-horizon reasoning with explicit intermediate structure and better orchestration rather than purely end-to-end black boxes. The overall trend is that practical AI progress increasingly depends on controllability, evaluation quality, and deployment-aware system design.

## Papers
### 1. When LLMs Stop Following Steps: A Diagnostic Study of Procedural Execution in Language Models
Date: 2026-05-01
URL: https://arxiv.org/abs/2605.00817
Summary: This paper studies whether language models actually execute a specified step-by-step procedure instead of merely landing on the right final answer. That matters because many real workflows depend on faithful process following, not just plausible outputs, and the reported drop-off on long procedures suggests a real reliability gap. The work is useful as a diagnostic lens for agents, reasoning systems, and any safety-critical prompting pattern that assumes models will follow instructions literally.

### 2. RunAgent: Interpreting Natural-Language Plans with Constraint-Guided Execution
Date: 2026-05-01
URL: https://arxiv.org/abs/2605.00798
Summary: RunAgent proposes a multi-agent execution framework that turns natural-language plans into constrained stepwise execution with validation, tool use, and error correction. This matters because one of the hardest problems in agent systems is preserving the flexibility of language while enforcing deterministic, checkable behavior. A framework that inserts constraints and per-step verification could make plan-driven agents more dependable in practical workflows.

### 3. To Call or Not to Call: A Framework to Assess and Optimize LLM Tool Calling
Date: 2026-05-01
URL: https://arxiv.org/abs/2605.00737
Summary: This paper analyzes the core agent decision of whether a model should call an external tool, especially for web search, using concepts such as necessity, utility, and affordability. This matters because many agent failures come from unnecessary, mistimed, or poorly integrated tool use rather than weak base-model reasoning alone. The proposed evaluative framework and lightweight estimators could help make tool-augmented systems both cheaper and more reliable.

### 4. Make Your LVLM KV Cache More Lightweight
Date: 2026-05-01
URL: https://arxiv.org/abs/2605.00789
Summary: LightKV compresses vision-token KV cache usage in large vision-language models by using prompt-aware cross-modal message passing during prefilling. This matters because multimodal systems can become impractical at deployment time when vision tokens consume too much GPU memory and inference budget. A method that preserves quality while materially reducing cache and compute cost could improve the deployability of vision-language agents.

### 5. Themis: Training Robust Multilingual Code Reward Models for Flexible Multi-Criteria Scoring
Date: 2026-05-01
URL: https://arxiv.org/abs/2605.00754
Summary: Themis builds multilingual code reward models and associated benchmarks that score not just functional correctness but multiple quality dimensions across languages. This matters because post-training and test-time ranking for coding systems are limited if they only optimize for whether code runs. Better reward models could improve coding agents on readability, maintainability, and broader engineering quality signals that matter in production.

### 6. Affordance Agent Harness: Verification-Gated Skill Orchestration
Date: 2026-05-01
URL: https://arxiv.org/abs/2605.00663
Summary: This work presents a closed-loop runtime for affordance grounding that routes among heterogeneous skills, stores evidence, retrieves episodic memories, and gates commitments through verification. This matters because many embodied and visual agents fail not from a single weak model but from brittle orchestration across perception and action modules. Verification-gated coordination is a promising systems approach for making open-world agents more robust under ambiguity and bounded compute.

### 7. Thinking in Text and Images: Interleaved Vision--Language Reasoning Traces for Long-Horizon Robot Manipulation
Date: 2026-05-01
URL: https://arxiv.org/abs/2605.00438
Summary: The paper introduces interleaved textual and visual reasoning traces as an explicit intermediate representation for long-horizon robot manipulation. This matters because text-only planning often misses geometry while pure latent control hides why a policy chooses a course of action. A model that reasons with both semantic subgoals and visual keyframes could make embodied policies stronger and easier to inspect.

### 8. What Makes a Good Terminal-Agent Benchmark Task: A Guideline for Adversarial, Difficult, and Legible Evaluation Design
Date: 2026-04-30
URL: https://arxiv.org/abs/2604.28093
Summary: This paper argues that terminal-agent benchmarks should be adversarial, difficult, and legible rather than written like prompts intended to help the model succeed. This matters because coding-agent claims increasingly depend on benchmark quality, and weak task design can systematically exaggerate capability. The paper is valuable as infrastructure work for a field that is moving quickly but still lacks enough disciplined evaluation practice.

### 9. Characterizing the Consistency of the Emergent Misalignment Persona
Date: 2026-04-30
URL: https://arxiv.org/abs/2604.28082
Summary: The authors study how consistently emergent misalignment behaviors and self-reported misalignment remain coupled after fine-tuning models on narrowly harmful domains. This matters because alignment failures are harder to detect and control when harmful tendencies generalize beyond the original training target. The paper sharpens the conversation around whether dangerous behavior forms a stable persona or a more fragmented and deceptive pattern.

## Themes
- Agent research is increasingly centered on process control: following plans, deciding when to use tools, and validating intermediate steps.
- Deployment-aware efficiency remains a major theme, especially for multimodal systems that strain GPU memory and serving budgets.
- Embodied AI is moving toward richer explicit reasoning structure instead of relying only on opaque end-to-end policies.
- Evaluation quality is becoming core infrastructure for coding and terminal agents as benchmarks begin to shape product claims.
- Alignment research continues to probe deeper behavioral failure modes rather than only measuring obvious harmful outputs.
