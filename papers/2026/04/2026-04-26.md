# AI Papers Digest — 2026-04-26
Selected recent arXiv papers.

## Overview
The most notable recent arXiv papers focus on making AI systems work better under practical constraints: too many tools, too little context, long interaction horizons, brittle multimodal grounding, and weak evaluation methods. Across agents, memory, reasoning, robotics, speech, and benchmark design, researchers are concentrating on the infrastructure around intelligent behavior rather than only scaling base models. The overall pattern is a push toward AI systems that are cheaper to run, easier to evaluate, and more reliable in open-ended real-world use.

## Papers
### 1. Tool Attention Is All You Need: Dynamic Tool Gating and Lazy Schema Loading for Eliminating the MCP/Tools Tax in Scalable Agentic Workflows
Date: 2026-04-23
URL: https://arxiv.org/abs/2604.21816v1
Summary: This paper argues that MCP-style eager schema injection creates a major hidden token and latency cost in multi-tool agents, and proposes gated tool selection plus lazy schema loading to reduce that overhead. That matters because many agent systems waste context budget before reasoning even begins. If the approach generalizes beyond the authors’ simulated benchmark, protocol efficiency could become just as important as larger context windows for production agents.

### 2. StructMem: Structured Memory for Long-Horizon Behavior in LLMs
Date: 2026-04-23
URL: https://arxiv.org/abs/2604.21748v1
Summary: StructMem proposes a hierarchical memory design that preserves relationships between events rather than storing isolated facts. This matters because long-running assistants need temporal and relational structure to support continuity, planning, and multi-hop reasoning. The reported gains alongside lower token usage and runtime make it especially relevant for deployed conversational and agentic systems.

### 3. Thinking with Reasoning Skills: Fewer Tokens, More Accuracy
Date: 2026-04-23
URL: https://arxiv.org/abs/2604.21764v1
Summary: The authors propose distilling reusable reasoning skills from expensive deliberation and retrieving those skills at inference time instead of solving every task from scratch. This matters because reasoning-heavy systems are increasingly constrained by inference cost, not only raw capability. A method that both reduces token spend and improves accuracy would have immediate practical value for large-scale AI products.

### 4. AEL: Agent Evolving Learning for Open-Ended Environments
Date: 2026-04-23
URL: https://arxiv.org/abs/2604.21725v1
Summary: AEL studies how agents can improve over many episodes by combining adaptive memory-retrieval policies with LLM-driven reflection on failures. This matters because persistent self-improvement remains one of the hardest open problems for autonomous agents in changing environments. The paper is also notable for showing a “less is more” result, where better use of experience outperforms additional architectural complexity.

### 5. MathDuels: Evaluating LLMs as Problem Posers and Solvers
Date: 2026-04-23
URL: https://arxiv.org/abs/2604.21916v1
Summary: MathDuels evaluates models as both adversarial problem authors and solvers instead of testing them only on fixed benchmark sets. This matters because many leading models are approaching saturation on static math evaluations, making real differences harder to detect. A dual-role setup could provide a more durable way to track reasoning progress as models become stronger.

### 6. When Prompts Override Vision: Prompt-Induced Hallucinations in LVLMs
Date: 2026-04-23
URL: https://arxiv.org/abs/2604.21911v1
Summary: This paper identifies prompt-induced textual priors as a major source of hallucination in large vision-language models and proposes a preference-optimization method to reduce the problem. That matters because multimodal systems are only useful when they stay grounded in actual visual evidence under complex instructions. Better measurement and mitigation of this failure mode are important for trustworthy assistants and perception systems.

### 7. Long-Horizon Manipulation via Trace-Conditioned VLA Planning
Date: 2026-04-23
URL: https://arxiv.org/abs/2604.21924v1
Summary: The authors present a modular method for long-horizon robotic manipulation that uses a task-management model and visual traces to guide execution and replanning. This matters because embodied AI often looks promising on short tasks but breaks down across multi-step workflows with accumulating errors. A more robust approach to decomposition and recovery is directly relevant to practical robot deployment.

### 8. Evaluation of Automatic Speech Recognition Using Generative Large Language Models
Date: 2026-04-23
URL: https://arxiv.org/abs/2604.21928v1
Summary: This paper explores using generative LLMs to evaluate ASR output semantically instead of relying primarily on word error rate. That matters because voice systems increasingly need metrics that reflect meaning and user perception rather than surface-form overlap alone. If LLM-based evaluation continues to track human judgment better, it could improve how speech products are tuned and benchmarked.

## Themes
- Agent performance is increasingly shaped by tool protocols, memory systems, and reuse of past reasoning.
- Evaluation is moving beyond static benchmarks toward adaptive, semantic, and harder-to-saturate testing.
- Efficiency remains central as token cost and runtime become hard constraints on reasoning systems.
- Multimodal reliability still depends on better grounding under strong textual priors.
- Long-horizon planning is emerging as a key frontier for embodied and open-ended AI systems.
