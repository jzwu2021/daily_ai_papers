# AI Papers Digest — 2026-04-27
Selected recent arXiv papers.

## Overview
The most interesting recent arXiv papers are less about scaling one giant model and more about the systems around intelligence: tool selection, reusable reasoning, inter-agent communication, safety certification, multimodal grounding, and long-horizon control. Researchers are trying to make AI systems cheaper to run, easier to evaluate, and more dependable in realistic settings where context is limited and errors compound over time. The day’s paper set suggests that practical AI progress now depends heavily on orchestration, evaluation design, and deployment discipline.

## Papers
### 1. Tool Attention Is All You Need: Dynamic Tool Gating and Lazy Schema Loading for Eliminating the MCP/Tools Tax in Scalable Agentic Workflows
Date: 2026-04-23
URL: https://arxiv.org/abs/2604.21816
Summary: This paper argues that MCP-style eager schema injection creates a large hidden token and latency cost in multi-tool agents, and proposes dynamic tool gating plus lazy schema loading to reduce that overhead. That matters because many production agents spend context budget on tool plumbing before they even begin reasoning. If this approach generalizes, protocol efficiency could become a major lever for improving agent quality and cost at scale.

### 2. Thinking with Reasoning Skills: Fewer Tokens, More Accuracy
Date: 2026-04-23
URL: https://arxiv.org/abs/2604.21764
Summary: The authors propose distilling reusable reasoning skills from expensive deliberation and retrieving them at inference time instead of solving each task from scratch. This matters because reasoning-heavy systems are increasingly constrained by token cost and latency, not just by raw capability. A method that lowers reasoning spend while improving accuracy would be immediately valuable for coding and math-focused AI products.

### 3. Learning to Communicate: Toward End-to-End Optimization of Multi-Agent Language Systems
Date: 2026-04-23
URL: https://arxiv.org/abs/2604.21794
Summary: This work studies multi-agent systems where the communication channel itself becomes trainable rather than fixed, using latent communication as a learnable component. That matters because many agent stacks focus on orchestration while treating message passing as an afterthought. Better communication primitives could make coordinated multi-agent reasoning more capable and more efficient.

### 4. Bounding the Black Box: A Statistical Certification Framework for AI Risk Regulation
Date: 2026-04-23
URL: https://arxiv.org/abs/2604.21854
Summary: The paper proposes a statistical certification framework for verifying whether high-risk AI systems satisfy explicit safety thresholds under emerging regulatory regimes. This matters because AI governance is moving faster than the technical methods used to prove that deployed systems are acceptably safe. Work like this can help translate broad policy requirements into testable operational criteria.

### 5. MathDuels: Evaluating LLMs as Problem Posers and Solvers
Date: 2026-04-23
URL: https://arxiv.org/abs/2604.21916
Summary: MathDuels evaluates models as both adversarial problem authors and solvers instead of testing them only on fixed benchmark sets. This matters because top models are nearing saturation on many static reasoning benchmarks, making progress harder to measure. A self-play style evaluation could produce harder, more durable tests of mathematical reasoning.

### 6. When Prompts Override Vision: Prompt-Induced Hallucinations in LVLMs
Date: 2026-04-23
URL: https://arxiv.org/abs/2604.21911
Summary: The authors identify prompt-induced textual priors as a major source of hallucination in large vision-language models and introduce a benchmark to study the problem. This matters because multimodal systems are only trustworthy when they stay grounded in what is actually visible, even under strong textual suggestions. Better diagnosis of this failure mode is important for assistants, perception systems, and safety-critical visual applications.

### 7. Long-Horizon Manipulation via Trace-Conditioned VLA Planning
Date: 2026-04-23
URL: https://arxiv.org/abs/2604.21924
Summary: This paper presents a modular framework for long-horizon robotic manipulation that uses a task-management model and visual traces to guide execution and replanning. This matters because embodied AI often looks strong on short tasks but degrades across multi-step workflows with compounding errors. A more robust planning-and-recovery scheme is directly relevant to practical robot deployment.

### 8. Evaluation of Automatic Speech Recognition Using Generative Large Language Models
Date: 2026-04-23
URL: https://arxiv.org/abs/2604.21928
Summary: This work explores using generative LLMs to evaluate ASR output semantically rather than relying mainly on word error rate. This matters because speech systems increasingly need metrics that reflect meaning and user perception, not just surface-form overlap. If LLM-based evaluation continues to track human judgment better, it could improve how voice products are benchmarked and tuned.

## Themes
- Agent quality is increasingly shaped by protocol design, memory, and communication efficiency.
- Evaluation is moving beyond static benchmarks toward adaptive, semantic, and harder-to-saturate methods.
- Efficiency matters not just in training, but in inference-time reasoning and tool use.
- Multimodal reliability still depends on better grounding under strong language priors.
- Safety and regulation are pushing researchers toward measurable certification frameworks, not just qualitative claims.
