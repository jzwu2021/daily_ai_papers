# AI Papers Digest — 2026-04-25
Selected recent arXiv papers.

## Overview
The most interesting recent arXiv papers are less about raw scale than about making AI systems work under real constraints: limited context, long horizons, noisy evaluation, and multimodal grounding failures. Across agents, memory, reasoning, robotics, speech, and evaluation, researchers are focusing on the structural bottlenecks that appear once models are expected to act, persist, and generalize in production-like settings. The overall trend is toward AI systems that are cheaper to run, easier to assess, and more reliable in open-ended use.

## Papers
### 1. Tool Attention Is All You Need: Dynamic Tool Gating and Lazy Schema Loading for Eliminating the MCP/Tools Tax in Scalable Agentic Workflows
Date: 2026-04-23
URL: https://arxiv.org/abs/2604.21816v1
Summary: This paper argues that MCP-style eager tool schema injection creates a major hidden token and latency cost in multi-tool agents, and proposes gating plus lazy loading as a remedy. That matters because scalable agent systems can waste context budget before reasoning even begins. If this approach generalizes beyond the authors’ simulated benchmark, protocol efficiency could become just as important as larger context windows for production agents.

### 2. StructMem: Structured Memory for Long-Horizon Behavior in LLMs
Date: 2026-04-23
URL: https://arxiv.org/abs/2604.21748v1
Summary: StructMem proposes a hierarchical memory design that preserves relationships between events instead of storing isolated facts. This matters because long-horizon assistants need temporal and relational structure to support continuity, planning, and multi-hop reasoning. The reported gains on reasoning quality alongside lower token and runtime costs make it especially relevant for deployed conversational agents.

### 3. Thinking with Reasoning Skills: Fewer Tokens, More Accuracy
Date: 2026-04-23
URL: https://arxiv.org/abs/2604.21764v1
Summary: The authors propose distilling reusable reasoning skills from lengthy deliberation and retrieving those skills at inference time instead of solving every task from scratch. This matters because reasoning-heavy systems are increasingly constrained by inference cost, not only by capability. A method that cuts token usage while improving accuracy would be valuable both economically and operationally for large-scale AI products.

### 4. AEL: Agent Evolving Learning for Open-Ended Environments
Date: 2026-04-23
URL: https://arxiv.org/abs/2604.21725v1
Summary: AEL studies how agents can improve across many episodes by combining adaptive memory-retrieval policies with LLM-driven reflection on failure patterns. This matters because persistent self-improvement remains one of the hardest open problems for autonomous agents operating in changing environments. The paper is also interesting for showing that extra architectural complexity can hurt, suggesting that better use of experience may matter more than adding more modules.

### 5. MathDuels: Evaluating LLMs as Problem Posers and Solvers
Date: 2026-04-23
URL: https://arxiv.org/abs/2604.21916v1
Summary: MathDuels turns models into both authors and solvers of adversarial math problems, rather than measuring them only on static benchmark sets. This matters because many top models are nearing saturation on fixed math evaluations, making it harder to see real capability differences. Dual-role evaluation could create a more durable way to track reasoning progress as models get stronger.

### 6. When Prompts Override Vision: Prompt-Induced Hallucinations in LVLMs
Date: 2026-04-23
URL: https://arxiv.org/abs/2604.21911v1
Summary: This paper identifies prompt-induced textual priors as a major source of hallucination in large vision-language models and proposes a preference-optimization approach to reduce the problem. That matters because multimodal systems are only useful when they stay grounded in actual visual evidence under complex instructions. Better tools for measuring and mitigating this failure mode are important for trustworthy AI assistants and perception systems.

### 7. Long-Horizon Manipulation via Trace-Conditioned VLA Planning
Date: 2026-04-23
URL: https://arxiv.org/abs/2604.21924v1
Summary: The authors present a modular approach for long-horizon robotic manipulation that uses a task-management model plus visual traces to guide execution and replanning. This matters because embodied AI often looks promising on short tasks but breaks down across multi-step, progress-dependent workflows. A more robust way to decompose and recover long sequences is directly relevant to practical robot deployment.

### 8. Evaluation of Automatic Speech Recognition Using Generative Large Language Models
Date: 2026-04-23
URL: https://arxiv.org/abs/2604.21928v1
Summary: This paper explores using generative LLMs to evaluate ASR output semantically, rather than relying mainly on word error rate. That matters because voice systems increasingly need metrics that reflect meaning and user perception, not just surface-form overlap. If LLM-based evaluation continues to match human judgments better, it could improve how speech products are tuned and benchmarked.

## Themes
- Agent quality is increasingly shaped by tool protocols, memory systems, and how prior experience is reused.
- Evaluation is moving beyond static benchmarks toward adaptive, semantic, and user-relevant testing.
- Efficiency remains central as token cost and runtime become hard constraints on reasoning systems.
- Multimodal robustness still depends on better grounding under instruction pressure.
- Long-horizon planning is becoming a key frontier for embodied and open-ended AI systems.
